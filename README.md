# Local Photo Engine (LPE)

**Local Photo Engine** is a high-performance, privacy-focused media management system designed to organize, search, and enjoy your photo library locally.

## Features
- **Semantic Search**: Find photos using natural language (e.g., "sunset at the beach").
- **Local First**: Your data stays on your machine.
- **AI-Powered**: Automatic face detection, object recognition, and scene classification.
- **Premium UI**: Designed with a sleek, modern aesthetic using Next.js and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
1. Clone the repository
2. Navigate to the UI directory:
   ```bash
   cd local-photo-engine-nextjs
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- **Frontend**: Next.js 15, React 19, Framer Motion
- **Styling**: Tailwind CSS
- **Local Storage**: Optimized for local filesystem integration
